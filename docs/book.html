MLtalk Book

This book is an official documentation of the MLtalk programming language. MLtalk is basically based on ML, but there's little bit extended original features. Take for example, goal-directed evaluation, if let error handling, etc.Let's go out to funny journey of the MLtalk!

01. Installation

You have to install interpreter of the MLtalk for use it. You can puts below install commands on your shell 


                    git clone https://github.com/KajizukaTaichi/MLtalk.git
                    cargo install --path ./Mltalk

If installing is finished successfully, you can puts just a mltalk binary name to start up MLtalk's REPL (Read-Evaluate-Print-Loop). You can use the MLtalk using REPL interactively looks like talking with computer is origin of it's called "MLtalk"

02. Function

Function is base of functional programming. In the MLtalk, you can define and apply function like below. 


                    let inc n = n + 1;
                    inc 5

That code returns 6 because 5 is applied by inc function to plus 1.
